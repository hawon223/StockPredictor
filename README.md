# StockPredictor
삼성전자(005930) 주가를 XGBoost로 예측하는 머신러닝 프로젝트입니다.  
한국 주가 데이터를 FinanceDataReader로 불러와, 과거 데이터를 기반으로 **다음 날 종가를 예측**합니다.
